ADP的学习之路01

发表于 2025-09-19 更新于 2025-09-24 分类于 智能体

假设你的目标是 “做一顿丰盛的晚餐”：

  • 只用 RAG： 你问它“红烧肉怎么做？”，它去翻菜谱，然后给你一个最正宗的做法。但它不会动手做。
  • 只用 Workflow： 你设置好一个流程：1. 洗菜 -> 2. 切菜 -> 3. 炒菜 -> 4. 装盘。它会把每一步都自动化执行，但如果你突然想改做清蒸鱼，整个流程就得重写。
  • 使用 Agent： 你直接告诉它“我饿了，想做点好吃的，冰箱里有排骨和土豆”。它会自己决定是做糖醋排骨还是土豆烧排骨，然后自己去搜索做法，控制炒菜机、电饭煲等工具，最后把菜端到你面前。
